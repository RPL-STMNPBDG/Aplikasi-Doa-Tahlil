Panduan / Tata Cara Aplikasi TAhlil-2022

1. User Interface Splash Screen
Ketika menekan icon aplikasi akan tampil gambar bergerak selama beberapa detik.

2. User Interface Menu Utama
Terdapat 4 menu dan satu info tambahan mengenai aplikasi.
	2.1 Menu Doa Tahlil
Ketika menekan menu Doa Tahlil akan tampil list urutan bacaan tahlil, dan ketika menekan salah satu item akan muncul pop up yang didalamnya terdapat bacaan ayat beserta terjemahnya. Pada bagian pojok juga terdapat fitur floating action button (Scroll to Top) untuk mengembalikan posisi /urutan scroll ke atas.
	2.2 Menu List Nama
Ketika menekan menu List Nama akan tampil form untuk create, update, dan delete nama jenazah, dan apabila telah terisi akan langsung tampil dibagian bawah tampilan form.
	2.3 Menu Shalat Jenazah
Ketika menekan menu Shalat Jenazah akan tampil list urutan tata cara melaksanakan shalat jenazah, untuk melihat isi dari judul item doa tersebut, klik icon down pada bagian sisi kanan judul item.
	2.4 Menu Shalat Ghaib
Ketika menekan menu Shalat Ghaib akan tampil list niat-niat shalat ghaib, untuk melihat isi dari judul item doa tersebut, klik icon down pada bagian sisi kanan judul item. Didalam isi doa juga terdapat fitur berupa audio bacaan niat yang dapat di play, pause, dan stop.
